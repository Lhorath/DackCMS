# DackCMS

> Custom PHP CMS with routing, templates, and admin-style workflows tailored for MacWeb Canada projects.

## Overview

DackCMS is a PHP content management system built for custom project deployments. It provides URL routing, template rendering, and admin workflows without the overhead of a general-purpose CMS.

## Requirements

- PHP and MySQL / MariaDB (as used by your deployment)
- A web server with URL rewriting if you use clean URLs (Apache `mod_rewrite` or Nginx equivalent)

## Setup

1. Deploy the project folder to your web host.
2. Configure database credentials per the project settings.
3. Point the document root at the project folder.
4. Ensure URL rewriting is enabled if using clean URLs.
